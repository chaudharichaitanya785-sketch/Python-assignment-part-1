# Student Grade Tracker — Part 1

## Overview

This project is a command-line Student Grade Tracker built using core Python concepts. It focuses on handling unstructured data, performing analysis, and generating meaningful summaries.

## Concepts Used

Lists, dictionaries, tuples
String methods (strip, title, split, replace)
Loops (for, while)
Conditional statements (if-elif-else)
Built-in functions (sum, max, min)
Input validation

## Tasks Implemented

Task 1 — Data Parsing and Profile Cleaning
Cleaned inconsistent student data
Converted roll numbers to integers
Parsed marks into lists
Validated student names
Displayed formatted student profile cards

Task 2 — Marks Analysis
Assigned grades based on marks
Calculated total and average
Identified highest and lowest scoring subjects
Built an interactive system to add new subjects
Handled invalid user inputs safely

Task 3 — Class Performance Summary
Calculated average marks for each student
Assigned pass or fail status
Generated a formatted class report
Identified class topper
Computed overall class average

Task 4 — String Manipulation Utility
Cleaned and formatted essay text
Counted keyword occurrences
Replaced words with formatted output
Split essay into sentences
Printed numbered sentences
